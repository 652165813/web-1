배운거 정리
ol은 목록에 번호를 붙여주는거, ul은 번호 없이 목록만 만들어주는거
ol이나 ul이라는 부모 태그 속에는 li라는 목록으로 만들어주는 자식 태그가 꼭 필요
!doctype html => 이 문서에는 html이 있다 라는 의미
html => html태그는 head와 body를 묶는 가장 상위의 태그
head => head태그는 본문을 설명하는 태그들을 묶기로 한 태그
title 오늘 일기 /title => 본문 제목이 무엇인지 설명하는 태그 title
meta charset="utf-8" => meta charset은 본문이 utf-8이라는 방식으로 저장되어 있다 라는 것을 설명하는 태그

body태그는 본문을 묶는 태그
a는 링크를 걸어두는 태그, a속에 href는 hyper text(링크) referencfe(참조)라는 의미
target=_"blank"는 링크를 눌렀을 때 새창으로 켜진다는 의미
img src=" 파일이름.jpg" width(크기)="100%" 크기 100%로 하면 페이지에 딱 맞게 됨
